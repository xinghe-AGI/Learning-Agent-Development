# 机器学习 Machine Learning

## 核心概念

**机器学习是让计算机根据数据调整模型参数，从而在未见过的新输入上进行预测、分类、生成或决策的方法。**

传统编程由人明确写出规则；机器学习则先定义模型、目标和训练方法，再让算法从数据中拟合规律。

```text
传统编程：数据 + 人工规则 -> 结果
机器学习：数据 + 目标答案/反馈 -> 模型
模型推理：新数据 + 模型 -> 预测或决策
```

## 主要学习范式

| 类型 | 训练信号 | 主要目标 | 典型任务 |
| --- | --- | --- | --- |
| [监督学习](<监督学习.md>) | 输入与标签 | 学习输入到输出的映射 | 分类、回归 |
| [无监督学习](<无监督学习.md>) | 无标签数据 | 发现结构、表示或分布 | 聚类、降维、异常发现 |
| [强化学习](<强化学习.md>) | 环境反馈与奖励 | 学习能提高长期回报的策略 | 控制、规划、序列决策 |

![监督学习、无监督学习与强化学习](<pictures/foundation-01-ml-paradigms.png>)

实际系统还会使用自监督学习、半监督学习、迁移学习等方法。许多 LLM 的预训练属于**自监督学习**：训练标签直接从原始文本中构造，例如用前文预测下一个 Token。

## 基本工作流程

1. **定义问题**：明确输入、输出、约束和评估指标。
2. **准备数据**：收集、清洗、标注并划分训练集、验证集和测试集。
3. **选择表示与模型**：决定特征、模型结构和基线方案。
4. **训练模型**：根据损失或反馈更新参数。
5. **评估与误差分析**：不仅看平均指标，还要分析失败类型与数据偏差。
6. **部署与监控**：观察真实数据上的质量、漂移、延迟和成本。
7. **迭代**：根据新数据和反馈继续改进。

## 机器学习与神经网络的关系

**神经网络是机器学习的一类模型，深度学习是使用多层神经网络进行表示学习的方法。**

```text
人工智能
└─ 机器学习
   ├─ 线性模型、树模型、核方法等
   └─ 神经网络
      └─ 深度学习
         └─ 大语言模型等应用方向
```

这是一张帮助理解的简化层级图，不代表所有 AI 都必须经过机器学习，也不代表 LLM 是深度学习的唯一产物。

## 常见误区

- **数据越多越好**：低质量、偏置或泄漏的数据会放大问题。
- **训练指标高就能上线**：还要验证泛化、鲁棒性、公平性和真实业务收益。
- **模型越大越好**：任务简单时，小模型通常更便宜、更快、更容易控制。
- **相关性等于因果性**：模型学到的预测规律不自动等于因果关系。
- **上线后模型不会变化**：环境和用户行为变化会造成数据漂移。
