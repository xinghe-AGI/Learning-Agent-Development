# 神经网络 Neural Network

## 什么是神经网络？

**神经网络是一类由多层可学习计算单元组成的机器学习模型。** 它受生物神经系统启发，但本质上是数学函数的组合，不是对人脑的完整模拟。

## 从单个神经元到网络

一个人工神经元通常完成三步：

1. 接收输入 `x`。
2. 使用权重 `w` 和偏置 `b` 计算加权和。
3. 经过激活函数得到输出。

```text
z = w1*x1 + w2*x2 + ... + b
output = activation(z)
```

激活函数为网络引入非线性，使多层网络能够表示复杂关系。常见激活函数包括 ReLU、Sigmoid、Tanh 和 GELU。

## 网络的基本层级

| 层级 | 作用 |
| --- | --- |
| 输入层 | 接收图像像素、特征数值或 Token 表示 |
| 隐藏层 | 对输入进行变换，逐步形成更适合任务的表示 |
| 输出层 | 生成类别概率、数值、Token 概率或其他结果 |

## 神经网络如何学习？

![神经网络训练循环](<pictures/foundation-02-neural-training-loop.png>)

1. **前向传播**：使用当前参数计算预测。
2. **计算损失**：衡量预测与训练目标之间的差距。
3. **反向传播**：利用链式法则计算各参数对损失的影响。
4. **参数更新**：优化器根据梯度调整权重和偏置。
5. **重复训练**：在多个批次和轮次中逐步降低损失。

**模型的能力分布在大量参数及其连接关系中。** 可以把参数理解为训练数据规律的压缩表示，但不能简单说“某个参数就保存了某条知识”。

## 为什么神经网络有效？

- **表示学习**：模型可以从原始数据中自动学习适合任务的特征。
- **函数逼近**：足够合适的结构和数据可以表示复杂非线性关系。
- **参数共享**：卷积、注意力等结构能复用计算模式，提高效率。
- **端到端优化**：多个处理步骤可以围绕同一目标共同训练。

## 常见限制

- 需要足够数据、算力和可靠评估。
- 训练过程可能不稳定，超参数会影响结果。
- 模型可能记住训练数据而不能泛化。
- 内部表示难以完整解释，不能只凭输出流畅度判断可靠性。
