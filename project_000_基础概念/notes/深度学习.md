# 深度学习 Deep Learning

## 什么是深度学习？

**深度学习是使用多层神经网络，从数据中学习层次化表示的一类机器学习方法。** “深度”主要指计算图中连续变换的层级较多，而不是用某个固定层数划出绝对边界。

## 核心思想：表示学习

传统机器学习经常依赖人工设计特征；深度学习则让模型围绕训练目标自动学习表示。

```text
传统方式：原始数据 -> 人工特征 -> 模型 -> 输出
深度学习：原始数据 -> 多层可学习表示 -> 输出
```

在图像任务中，早期层可能更容易响应边缘和纹理，后续层逐步组合出部件与语义；在语言模型中，Token 表示经过多层变换后逐步融合上下文信息。

## 深度学习带来的变化

- **减少手工特征工程**：模型可以从原始或弱处理数据中学习表示。
- **端到端训练**：感知、表示和预测可以围绕同一损失共同优化。
- **规模化能力**：数据、参数和算力增长后，模型可以处理更复杂的模式。
- **迁移与预训练**：先在大规模数据上学习通用表示，再适配下游任务。

## 常见架构

| 架构 | 主要特点 | 常见方向 |
| --- | --- | --- |
| CNN | 利用局部连接和参数共享处理空间结构 | 图像、视觉感知 |
| RNN / LSTM | 按序列递归传递状态 | 早期语音、文本与时间序列 |
| Transformer | 使用注意力和前馈网络进行序列建模 | LLM、多模态、视觉与语音 |
| Autoencoder | 学习压缩表示并重建输入 | 表示学习、降维、生成 |

## 深度学习与 LLM 的关系

LLM 是深度学习在语言与序列建模方向的代表性应用。大多数主流生成式 LLM 使用 Transformer，并通过大规模预训练学习语言模式，再经过指令微调、偏好优化或其他后训练提升可用性。

## 局限性

- 训练成本与能耗较高。
- 对数据质量、覆盖范围和偏差非常敏感。
- 很难仅凭内部参数解释具体输出原因。
- 在分布外数据上可能自信地犯错。
- 性能提升不只依赖增加层数，还取决于架构、优化、数据和评估。
