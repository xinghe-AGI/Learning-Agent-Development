## 知识关系图

```mermaid
flowchart LR
    AI[人工智能 AI] --> ML[机器学习]
    ML --> SL[监督学习]
    ML --> UL[无监督学习]
    ML --> RL[强化学习]
    ML --> NN[神经网络]
    NN --> DL[深度学习]
    DL --> LLM[大语言模型 LLM]
    LLM --> Token[Token]
    LLM --> Transformer[Transformer]
    Transformer --> Attention[自注意力]
    LLM --> NTP[下一个 Token 预测]
    NTP --> AR[自回归生成]
    Prompt[Prompt] --> LLM
    LLM --> App[AI 应用]
    App --> LangChain[LangChain]
    App --> LangGraph[LangGraph]
    App --> LangSmith[LangSmith]
```

这张图需要注意两点：

- **层级关系不等于唯一实现。** 不是所有 AI 都依赖机器学习，也不是所有神经网络都属于深度学习。
- **训练目标和生成方式不是一回事。** [下一个 Token 预测](<下一个词预测训练目标.md>)描述模型如何学习，[自回归生成](<自回归生成方式.md>)描述模型如何一步步输出。
