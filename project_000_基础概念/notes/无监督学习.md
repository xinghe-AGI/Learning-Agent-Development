# 无监督学习 Unsupervised Learning

## 什么是无监督学习？

**无监督学习在没有人工标签的情况下，从数据中发现结构、分布或更紧凑的表示。** 它通常不会直接给出“标准答案”，而是帮助我们理解数据中可能存在的群组、方向和异常。

## 常见任务

| 任务 | 目标 | 常见方法 |
| --- | --- | --- |
| 聚类 | 把相似样本分组 | K-means、层次聚类、DBSCAN、GMM |
| 降维 | 用更少维度保留主要结构 | PCA、t-SNE、UMAP |
| 异常发现 | 找到明显偏离整体分布的样本 | Isolation Forest、局部离群方法 |
| 表示学习 | 学习便于下游任务使用的特征 | 自编码器、对比学习 |

## 一个最小聚类示例

```python
from sklearn.cluster import KMeans

# 两个特征：购买金额、购买次数
X = [[100, 5], [120, 6], [200, 20], [210, 22], [30, 1], [35, 2]]

model = KMeans(n_clusters=2, random_state=42, n_init="auto")
labels = model.fit_predict(X)
print(labels)
```

聚类编号本身没有业务含义。还需要回看每一组的特征，才能把它解释为“高频高消费”或“低频低消费”等业务标签。

## 如何理解结果？

- **聚类是相似性分组，不是真实世界的天然类别。** 换一种距离、特征或簇数，结果可能变化。
- **降维图适合探索，不等于证明。** 二维可视化会丢失信息，图上分开不代表原空间中一定存在清晰边界。
- **异常不等于错误或欺诈。** 它只是“与多数样本不同”，仍需要人工判断。

## 适合使用的场景

- 数据没有可靠标签，但需要先探索结构。
- 想发现用户群体、主题或行为模式。
- 需要压缩特征、可视化高维数据。
- 需要为后续监督学习准备表示或候选标签。
